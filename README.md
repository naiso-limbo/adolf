「单词磨耳朵」是一款好玩又有效的英语单词听学软件，让你磨着耳朵就把单词背了，是提高英语听力水平的好伙伴。

# 你能做什么？
可以选择不同的词典词汇，听着单词的拼读和翻译，逐渐养成单词记忆。\n无论在运动跑步、上下班通勤，还是无聊的时候都可以打开「单词磨耳朵」，就能让你不知不觉就把单词背了。\n如果你还有入睡困难，也可以用来做白噪音，辅助入睡。
